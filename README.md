##instalacion
descargar los archivos y ejecutarapp.py

  -Instalar bibliotecas necesarias:
  ```bash
  pip install -r requirements.txt
  ``

  -Creamos *.env* y lo configuramos:
  ```bash
  cp envExample .env
  ``

  -Ejecutamos
  ```bash
  python app.py
  ``
